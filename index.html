I’m Philipp Steinmann,

A student in New York City, of German and Greek origins.

My interests include programming, literature, and language.

I’m a perfectionist, which is why I’m up at this ungodly hour right now tweaking my website and why I spend countless hours configuring, enhancing, and learning topics and tools from text editors to cryptography. I don’t do things half-way when I’m passionate about them.

Programming

Ever since I learned Python a few years back, I’ve been mesmerized by programming and the power it holds; it is astounding that I can tell a machine what to do to solve complex problems, using abstract symbolic manipulations that were unheard of a few generations ago.

Web Development

I do web development for my school’s robotics team, where I am vice president of marketing. I love using Ruby on Rails, AJAX, and good design to create a smooth user experience, though I’m not the fastest, since I like to do it slowly and with care. I’m immersed in web startup culture through Hacker News, and find the rapid work cycle of web development so appealing that I could see myself doing stuff like this in the future.

Linux

I switched to using Linux as my main operating system after I realized that web development is pretty much impossible using Windows. Since, then I’ve loved it; running the smooth Crunchbang 11 on an SSD-powered laptop, I've been learning how to do a lot of cool things over the terminal (such as reading emails), and wield the power of awesome programs such as vim and sed.
