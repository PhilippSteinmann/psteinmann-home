I'm Philipp Steinmann,

A student in New York City, of German and Greek origins.

My interests include programming, literature, and language.

Programming

Ever since I learned Python a few years back, I've been mesmerized by programming and the power it holds; the fact that I can tell a machine what to do in order to solve a problem is something that astounds me every time I think about it.

Web Development

I do web development for my school's robotics team, where I am vice president of marketing. This year, with a new and modern-minded team, we migrated away from PHP and started using Ruby on Rails and meticulous version control. I love the rapid work cycle of web development, and, after careful observing of the startup scene through Hacker News, could see myself doing cutting-edge stuff like this in the future.

Linux

I switched to using Linux as my main operating system after I realized that web development is pretty much impossible using Windows. Since, then I've loved it; the Debian-based Crunchbang 11 runs incredibly smoothly on a laptop with an SSD. I've also been learning how to use the Bash shell, and awesome commands like sed.
