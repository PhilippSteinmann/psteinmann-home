I'm Philipp Steinmann,

A student in New York City, of German and Greek origins.

My interests include programming, literature, and language.

Programming

Ever since I learned Python a few years back, I've been mesmerized by programming and the power it holds; the fact that I can tell a machine what to do in order to solve a problem is something that astounds me every time I think about it.

Web Development

I do web development for my school's robotics team, where I am vice president of marketing. I love using Ruby on Rails, AJAX, and good design to create a smooth user experience. After immersing myself in the web startup community through Hacker News, I find the rapid work cycle of web development appealing, and could see myself doing cutting-edge stuff like this in the future.

Linux

I switched to using Linux as my main operating system after I realized that web development is pretty much impossible using Windows. Since, then I've loved it; I'm currently running Crunchbang 11, which is incredibly smooth on my SSD-powered laptop. I've also been learning how to do a lot of things over the terminal (such as reading emails), and wield the power of awesome programs like vim and sed.
